Логика

После получения Workflow необходимо:

1. Получить все шаги Workflow через WorkflowStepService используя метод list_workflow_steps(workflow.id)

2. Шаги уже должны приходить отсортированными. Никакой дополнительной сортировки делать не нужно.

3. Для каждого шага:
prompt = prompt_builder.build_prompt(step.prompt)
response = await llm_provider.generate(prompt)

4. Последний полученный response сохранить в переменную final_response

5. После окончания цикла: WorkflowRun.result должен стать

{
    "response": final_response
}

Все остальные поля оставить без изменений. Статусы running/completed должны работать как сейчас.
