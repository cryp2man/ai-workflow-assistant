# ENV-002 - Run Development Environment

## Контекст

Проект AI Workflow Assistant.

Уже реализовано:

- Project Setup
- FastAPI
- Application Core
- API Structure
- Health Endpoint
- Database Foundation
- Alembic Setup
- Docker Environment

Архитектура проекта утверждена.

---

## Цель

Полностью проверить локальное окружение разработки.

После выполнения должны корректно работать:

- Docker
- PostgreSQL
- pgAdmin
- FastAPI
- Alembic

---

## Необходимо выполнить

### 1.

Проверить наличие Docker Desktop.

Если Docker не установлен - остановиться и сообщить об этом.

Не пытаться устанавливать Docker автоматически.

---

### 2.

Создать файл

.env

на основе

.env.example

если он отсутствует.

Использовать безопасные значения для локальной разработки.

---

### 3.

Запустить контейнеры

docker compose up -d

---

### 4.

Проверить

docker compose ps

Все сервисы должны быть в состоянии healthy/running.

---

### 5.

Проверить PostgreSQL

Подключение должно успешно выполняться.

---

### 6.

Проверить pgAdmin

Убедиться, что сервис доступен.

Не создавать сервер вручную.

---

### 7.

Проверить Alembic

Выполнить

alembic current

Команда должна завершиться успешно.

Отсутствие миграций является корректным результатом.

---

### 8.

Проверить FastAPI

Запустить

uv run uvicorn src.main:app --reload

Проверить:

GET /api/v1/health

Swagger

---

## Не делать

НЕ создавать:

- модели
- миграции
- таблицы
- Dockerfile
- дополнительные сервисы

---

## Acceptance Criteria

После выполнения:

✔ Docker работает

✔ PostgreSQL работает

✔ pgAdmin работает

✔ FastAPI работает

✔ Alembic current выполняется

✔ Health endpoint отвечает

---

## Отчет

Предоставить:

1. Какие команды были выполнены.
2. Какие проверки пройдены.
3. Какие URL доступны.
4. Были ли ошибки.
5. Что рекомендуется сделать дальше.

Если возникла ошибка - остановиться и подробно описать её, не исправляя без отдельного задания.
