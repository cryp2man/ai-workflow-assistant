PROMPT-001 - создать Prompt Builder

Контекст проекта

В проекте уже существует

ExecutionEngine

LLM Layer

(BaseLLMProvider и OpenAIProvider)

Перед подключением реального OpenAI необходимо создать слой формирования промптов.

========================================

Работаем строго по принципу Surgical.

Разрешается создать только

src/prompts/

и файлы внутри этой директории.

========================================

Не изменять

Engine

LLM

Repositories

Services

Models

Schemas

API

Alembic

Tests

========================================

Создать структуру

src/prompts/

с файлами

__init__.py

builder.py

========================================

В builder.py создать класс

PromptBuilder

========================================

Реализовать метод

build_prompt(
    self,
    prompt: str,
) -> str

Пока метод должен просто вернуть переданный prompt без изменений.

Никаких шаблонов.

Никакой логики.

Никакого форматирования.

========================================

В __init__.py экспортировать

PromptBuilder

========================================

После выполнения проверить

import src.main

uv run ruff check src/prompts

========================================

В отчёте указать

1. Какие файлы созданы.

2. Почему PromptBuilder выделен в отдельный слой.

3. Почему build_prompt() пока просто возвращает строку.

4. Какую роль PromptBuilder будет играть позже.

5. Проходит ли импорт.

6. Результат Ruff.

Никаких дополнительных улучшений не выполнять.
