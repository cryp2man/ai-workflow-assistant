========================================
Цель
========================================

Подготовить базовую инфраструктуру тестовой базы данных.

Пока НЕ писать тесты.

========================================

Разрешается изменить только:

tests/conftest.py

========================================

Необходимо реализовать:

1. Создать async SQLAlchemy Engine для тестовой базы.

Использовать:
create_async_engine()

2. Создать

TestingSessionLocal

через

async_sessionmaker()

3. Создать fixture

db_session

которая

- открывает AsyncSession
- делает yield
- после завершения закрывает сессию

Без rollback.

Без truncate.

Без очистки таблиц.

Только инфраструктура.

4. DATABASE_URL брать НЕ из production Settings.

Использовать отдельную строку подключения внутри conftest.py

например

postgresql+asyncpg://user:password@localhost:5432/test_db

(если test_db отсутствует - не создавать её автоматически)

========================================

Запрещено

- удалять таблицы
