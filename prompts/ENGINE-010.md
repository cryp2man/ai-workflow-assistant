=========================
Задача
=========================

Необходимо добавить измерение времени выполнения каждого WorkflowStep.

Использовать:

import time

Перед вызовом

await self.llm_provider.generate(...)

сохранить

start_time = time.perf_counter()

После получения response вычислить

elapsed = time.perf_counter() - start_time

После успешного выполнения шага добавить лог:

Step completed: {step.step_order} ({elapsed:.3f}s)

То есть существующий лог

Step completed: X

заменить на

Step completed: X (0.532s)

При ошибке ничего дополнительно измерять не нужно.

Только успешные шаги.
