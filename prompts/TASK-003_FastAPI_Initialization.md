# TASK-003 - FastAPI Initialization

## Контекст

Проект AI Workflow Assistant.

Стек уже утвержден.

Ничего не менять.

Работаем строго поэтапно.

## Цель

Создать минимальное приложение FastAPI.

## Необходимо выполнить

1. Создать точку входа приложения.

2. Создать экземпляр FastAPI.

3. Добавить:

- название проекта;
- описание;
- версию API.

4. Настроить запуск через uv.

5. Проверить успешный запуск сервера.

6. Swagger должен открываться по адресу:

/docs

7. ReDoc должен открываться по адресу:

/redoc

## Ограничения

НЕ создавать:

- модели БД;
- SQLAlchemy;
- Alembic;
- роутеры;
- сервисы;
- бизнес-логику;
- Docker.

## Критерии готовности

После выполнения должно работать:

uv run uvicorn src.main:app --reload

После запуска должны открываться:

http://127.0.0.1:8000/docs

http://127.0.0.1:8000/redoc

Без ошибок.
