REP-003 - WorkflowRepository CRUD.

ВАЖНО.

Работаем строго по принципу Surgical.

========================================

Разрешается изменить только

- src/repositories/workflow_repository.py

========================================

Запрещается менять

- Models
- Services
- API
- Dependencies
- Alembic
- Database
- Schemas
- Tests

========================================

Цель

Реализовать CRUD-операции для WorkflowRepository по аналогии с UserRepository.

========================================

Необходимо реализовать методы:

- create(workflow: Workflow) -> Workflow
- get_by_id(workflow_id: int) -> Workflow | None
- list() -> list[Workflow]
- list_by_user(user_id: int) -> list[Workflow]
- delete(workflow: Workflow) -> None

========================================

Требования

Использовать SQLAlchemy 2.0 стиль.

Использовать:

- select()
- where()
- session.execute()
- scalar_one_or_none()
- scalars().all()

В create() и delete() использовать только:

await session.flush()

Никаких commit()
Никаких rollback()
Никаких refresh()

========================================

Запрещается

- менять BaseRepository
- добавлять Generic
- использовать TypeVar
- писать SQL вручную
- менять Workflow model

========================================

После выполнения проверить:

- import src.main
- uv run ruff check src/repositories/workflow_repository.py

5. Почему commit отсутствует.
