TEST-001 - Pytest Foundation.

ВАЖНО.

Работаем по принципу Surgical.

НЕ менять:

- src/
- alembic/
- compose.yaml
- модели
- сервисы
- репозитории
- API
- миграции
- Docker

========================================
Цель
========================================

Подготовить инфраструктуру тестирования проекта.

Без написания самих тестов.

========================================

Разрешается изменить только:

pyproject.toml

tests/

========================================

Необходимо создать следующую структуру:

tests/
    __init__.py
    conftest.py

========================================

В pyproject.toml

добавить минимальную конфигурацию pytest.

Если pytest уже настроен - ничего не менять.

========================================

В conftest.py

пока НЕ создавать фикстуры БД.

Нужно только оставить базовый файл с комментарием:

"""
Project test fixtures.

Database fixtures
API fixtures
Repository fixtures

будут добавляться постепенно.
"""

========================================

Проверить

uv run pytest

Если тестов нет - это считается успешным результатом.

========================================

Не устанавливать новые библиотеки.

Не добавлять pytest-asyncio.

Не добавлять httpx.

Не писать тесты.

Не создавать фикстуры БД.

========================================

В отчёте указать:

1. Какие файлы созданы.

2. Какие файлы изменены.

3. Что добавлено в pyproject.toml.

4. Результат
uv run pytest

5. Почему пока нет тестов. Никаких дополнительных улучшений не делать.
