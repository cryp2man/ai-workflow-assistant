TEST-007 - Async Engine Lifecycle.

ВАЖНО.

Работаем строго по принципу Surgical.

========================================

НЕ менять

- src/
- API
- Repository
- Service
- Models
- Alembic
- compose.yaml

========================================

Разрешается изменить только

tests/

========================================

Цель

Исправить проблему

RuntimeError: Event loop is closed

и

InterfaceError:
another operation is in progress

при последовательном выполнении async тестов.

========================================

Необходимо

Исправить исключительно инфраструктуру тестирования.

Не менять бизнес-код.

========================================

Разрешается

- изменить фикстуры pytest
- изменить scope фикстур
- изменить lifecycle test_engine
- использовать pytest-asyncio рекомендации
- использовать loop_scope, если это необходимо

========================================

Запрещается

- менять src/
- менять Repository
- менять Service
- менять API
- менять Models

========================================

После исправления выполнить

uv run pytest

========================================

В отчёте указать

1. Что являлось причиной ошибки.

2. Какие фикстуры изменены.

3. Как теперь создаётся engine.

4. Как теперь закрывается engine.

5. Удалось ли выполнить все тесты.

6. Если тесты всё ещё падают - указать единственную причину.

Никаких дополнительных улучшений не делать.
