REP-004 - WorkflowRunRepository

Контекст проекта

FastAPI
SQLAlchemy 2.x
Repository Pattern

В проекте уже существуют:

- UserRepository
- WorkflowRepository

Необходимо реализовать аналогичный репозиторий для WorkflowRun.

========================================

Работаем строго по принципу Surgical.

Разрешается изменить только

src/repositories/workflow_run_repository.py

src/repositories/__init__.py

Если файла workflow_run_repository.py нет - создать его.

========================================

Создать класс

WorkflowRunRepository

унаследованный от BaseRepository.

========================================

Реализовать методы

create(
    workflow_run: WorkflowRun,
) -> WorkflowRun

get_by_id(
    run_id: int,
) -> WorkflowRun | None

list() -> list[WorkflowRun]

list_by_workflow(
    workflow_id: int,
) -> list[WorkflowRun]

delete(
    workflow_run: WorkflowRun,
) -> None

========================================

Следовать полностью стилю

UserRepository

WorkflowRepository

========================================

Использовать

select()

where()

session.execute()

scalar_one_or_none()

scalars().all()

session.add()

session.delete()

session.flush()

========================================

commit()

rollback()

refresh()

использовать запрещено.

========================================

После выполнения

проверить

import src.main

uv run ruff check src/repositories/workflow_run_repository.py
