# TASK-002 - Project Setup

## Context

Мы разрабатываем проект AI Workflow Assistant.

Стек проекта уже утвержден и не должен изменяться.

Используем:

- Python 3.13
- uv
- FastAPI
- PostgreSQL
- SQLAlchemy 2.x
- Alembic
- Pydantic v2
- Docker Compose
- Pytest
- Ruff
- Black

Работа ведется на Windows 11.

---

## Goal

Подготовить проект к разработке.

---

## Необходимо выполнить

1. Инициализировать проект с помощью uv.
2. Создать виртуальное окружение.
3. Создать файл pyproject.toml.
4. Добавить все необходимые зависимости.
5. Создать базовую структуру проекта.
6. Создать .gitignore.
7. Создать .env.example.
8. Настроить Ruff.
9. Настроить Black.
10. Настроить Pytest.
11. Не писать бизнес-логику.
12. Не создавать API.
13. Не писать Dockerfile.
14. Не создавать модели базы данных.

---

## Требуемая структура проекта

src/

api/

core/

db/

models/

schemas/

services/

bot/

tests/

docs/

architecture/

prompts/

---

## Acceptance Criteria

После выполнения задачи должны выполняться следующие условия:

- проект открывается без ошибок;
- uv установлен и используется;
- все зависимости устанавливаются одной командой;
- виртуальное окружение создано;
- Ruff работает;
- Black работает;
- Pytest запускается;
- структура проекта соответствует заданной.

---

## Ограничения

Не менять стек технологий.

Не добавлять дополнительные библиотеки без необходимости.

Не реализовывать функциональность следующих этапов проекта.

Следовать принципу KISS (Keep It Simple).
