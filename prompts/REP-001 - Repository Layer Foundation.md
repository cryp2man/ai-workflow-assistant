
## Цель

Создать фундамент Repository Layer для проекта.

Важно:

- НЕ менять API.
- НЕ менять модели.
- НЕ менять миграции.
- НЕ менять сервисы.
- НЕ менять эндпоинты.
- НЕ менять поведение приложения.

Repository Layer пока только создается.

---

## Структура

Создать новую директорию:

src/repositories/

Создать файлы:

- __init__.py
- base.py
- user_repository.py
- workflow_repository.py

---

## BaseRepository

Создать базовый класс BaseRepository.

Он должен принимать в конструкторе:

AsyncSession

Базовый класс пока НЕ должен реализовывать CRUD полностью.

Разрешается добавить только инфраструктуру:

- сохранение session
- базовые protected helper-методы (если действительно нужны)

Не делать Generic Repository.

Не использовать TypeVar.

Не использовать Generic[T].

Не использовать metaclass.

Максимально простой код.
