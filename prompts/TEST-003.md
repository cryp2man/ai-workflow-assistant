========================================

Цель

Подготовить pytest к работе с async кодом проекта.

Без написания тестов.

========================================

Необходимо

1. Добавить pytest-asyncio в dev dependencies. Только если его ещё нет.

2. Настроить pytest через pyproject.toml в соответствии с современным режимом SQLAlchemy 2.x.

Использовать

asyncio_mode = "auto"

3. Никаких тестов не писать.

4. conftest.py не менять.

========================================

Проверить

uv sync

после изменения зависимостей.

Затем

uv run pytest

Ожидается

0 tests collected

без ошибок.
